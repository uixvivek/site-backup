IRCTC Redesign- Search & Booking Flow

UI/UX Design Figma Product Design

Project Overview

IRCTC is one of India’s most used platforms for train ticket booking, but the experience often feels cluttered, overwhelming, and inconsistent. This project is a UI/UX redesign focused on improving the core booking flow- making it faster, cleaner, and easier for users to complete the main task: booking a train ticket.

The Challenge

The biggest issue with IRCTC is not lack of features- it’s too many features competing for attention. Users face visual clutter, confusing hierarchy, and too many steps for simple actions like searching trains and checking availability.

My Role & Process

This was a self-initiated UX case study where I worked as the product designer end-to-end. I analyzed the existing IRCTC flow, identified friction points, redesigned key screens, and created a smoother flow using strong information hierarchy and cleaner UI patterns.

Key Features

  • Feature 1 - Simplified layout with better spacing and hierarchy.
  • Feature 2 - Removed unnecessary distractions and focused on the primary task.
  • Feature 3 - Screens feel consistent and predictable for the user

Design Decisions

My redesign focuses on clarity over decoration. I used a minimal visual system, stronger typography hierarchy, and consistent spacing. I also kept the UI familiar so users don’t feel like they are learning a completely new app- it still feels like IRCTC, but modern and usable.

Results & Impact

This redesign makes the main booking journey feel more direct and less frustrating. The biggest improvement is cognitive load reduction- users can focus on the task instead of fighting the interface.

Goal: Make the IRCTC booking flow feel simple, clean, and trustworthy- without removing what users already know.
